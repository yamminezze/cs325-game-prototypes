Breakout Unchained!

  • Originality: A breakout game without being stuck of the horizontal axis.
  • Technical Merit: The ball moves faster than most breakout games since you now have the freedom of moving the paddle vertically as well.
  • Prototype Postmortem: What did you learn from this prototype? What was the easiest or hardest part of making it?
  • Prototype Assets: Most of the assets are from an expample Phaser game called "Breakout". I changed the background image using a Google image. Source: https://goo.gl/images/aaydBJ
    • Bouncing Phaser image: Phaser logo
  • Prototype Closest Other Game: The example Phaser game "Breakout" is the game I used to make my version. Much of the code is similar except for what is needed to allow the paddle to move vertically and all the changes that come with that as well as changing the velocity of the ball. Since the difficulty has changed, the point values changed as well.
  • High Concept: Destroy all the bricks before running out of lives
  • Theme: ?
  • Mandated Variety: It uses mouse click and movement input. It can become a fast paced game at certain points. If the ball hits the paddle directly in the center, there is some randomness to where the ball goes.
  • Prototype Goal: Ball bouncing physics and user input measurment.
  • Player Experience Goals: Having the satisfaction of beating the entire level (or multiple levels) and seeing their score at the end
  • Gameplay: Click to start and launch the ball. You use the mouse to control where the paddle is. If the ball falls below your paddle, you lose a life. Lose 3 lives and its game over. You get points for each brick you destroy and a big bonus if you destroy them all. Good Luck!
  • Strategies: Knowing where to bouce the ball so that it destroys as many bricks as possible before returning to the paddle.
  • Story/Setting/Premise: This game doesn't have much story. It takes place in space, and you're a paddle destroying bricks. Maybe in the future it could be upgraded.
  • Target Audience: This game is great for all ages and levels of experience.
  • Play Time: A couple of minutes. Can be longer if player is highly skilled. -- Version by Yammin Ezzedine
